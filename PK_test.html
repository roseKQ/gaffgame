Primary Keys, Foreign Keys, Composite Keys.

We use keys to create relationships between tables in a database. This makes it easier to retrieve records from the database tables.

There are a number of different types. In this section, we will be learning how to use Primary Keys A primary key uniquely identifies each record in a database table. They must contain unique values. Each table can have only ONE primary key., Foreign Key A foreign key points to a primary key in another table. or Composite Key A composite key is a combination of two or more columns in a table that can be used to uniquely identify each row..
